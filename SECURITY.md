# Security Policy

## Supported Versions

Versions currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 4.x     | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

You can report security issues directly on GitHub. Reports are confidential.

Any reported security issues will be given top priority.
